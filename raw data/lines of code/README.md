# Количество строк кода
- loc_collect.sh - bash-скрипт, который обращается к директории с PostgreSQL, производит переключение на ветки кода соответствующие первым дням года. После этого запускает утилиту github.com/AlDanial/cloc и выводит результат на экран
- loc_out.txt - вывод скрипта loc_collect.sh > loc_out.txt
- loc_row_data.csv - обработанные в табличный вид данные из loc_out.txt, плюс добавлена информацию о строках кода из проекта Postgres 4.2 (1994 год)

Перед запуском необходимо клонировать https://github.com/AlDanial/cloc
